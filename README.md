# NoBroker Clone

A Progressive Web App (PWA) for real estate property listings, similar to NoBroker. Built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Property search and listings
- Buy, rent, and sell options
- PWA capabilities (offline support, installable)
- Responsive design

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages
- `public/` - Static assets and PWA manifest
- `next.config.ts` - Next.js configuration with PWA setup

## Technologies Used

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- next-pwa for PWA functionality
